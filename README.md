## О последнем обновлении

Начата разработка текстового энкодера. Задумка следующая:
- Входящие значения должны быть упакованы в структуру input
- Первым шагом в текстовом энкодере структура токенизируется
- Затем по токенам выбираются эмбединги
- Эмбединги дополнительно складываются с вектором позиций
- Позиционированный эмбединг отправляется на много-головый блок внимания
- В блоке внимания параллельно начинается вычисление векторо Q, K, V
- Вычисляется оценка схожести по QK
- QK нормализуется и проходит через softmax затем умножается на V
- Результаты параллельных вычилений кокатенируются и умножаются на WO
- Полученная матрица складывается с матрицей переданной на блок внимания, затем нормализуется
- Итоговое пространство признаков с помощью одного слоя нейросети сначала раздувается, затем сжимается обратно
